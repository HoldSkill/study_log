### 1.java特性和优势

- 简单性
- 面向对象
- 可移植性
- 高性能
- 分布式
- 动态性
- 多线程
- 安全性
- 健壮性

### 2. Java三大版本

Write once ,run anywhere (跨平台)

- JavaSE:标准版 （桌面程序，控制台开发），最核心，最基本
- javaMe:嵌入式开发(手机，小家电)
- JavaEE:企业级开发(web端，服务器开发)

### 3. JDK,JRE,JVM

- JDK: Java development kit  Java开发者工具
- JRE：Java run environment Java运行环境
- JVM：Java Virtual Machine Java虚拟机
